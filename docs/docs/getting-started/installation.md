## Requirements

**Note**: Only the core requirements are listed

### Frontend

- Yarn >= v2

### Backend

- Node >= 14.17.0

## Installation

```
git clone https://github.com/GDSCUTM-CommunityProjects/UTimeManager
```
### Frontend

```
cd client
yarn install 
yarn run start
```

### Backend

```
cd backend
yarn install
yarn run start
```
