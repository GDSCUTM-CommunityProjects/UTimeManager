<Endpoint>POST /feedback</Endpoint>: Create a feedback with provided feedback details <br></br>

```json
{
    "body": string,
    "user_id": String,
    "task_id": String,
    "satisfaction": Number,
    "createDate": String,
}
```

This is a **protected route**, a **valid JWT is required** in the header field
